структура роли
/defaults - важная папка в которой находятся все переменные которые будет использовать роль
Эти переменные могут быть перезаданы уровнем выше в плэйбуке. Это как дефолтные значения
/files - тут находятся все файлы которые могут быть куда то скопированы. тут конкретно для
роли
/handlers - обработчики событий, их можно вызвать чтобы какие-либо действия совершить
например, рестартнуть сервис. Чаще всего это 
/meta - инфа о роли. справочная.
/tasks - директория где хранятся таски для роли
/templates - там где шранятся шаблоны которые превращаются потом в конфиг
/tests - опциональный параметр. хранятся тесты для роли этой
/vars - перечень переменных. но их нельзя переопределить нигде. системные переменные
например. не часто используется
Еще удобно в именах тасков когда пишем роль использовать имя роли затем | и уже что
делает эта роль. Удобно при дебаге
